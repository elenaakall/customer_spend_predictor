# 🛒 Customer Spend Predictor

This repository contains the deployment code for a Machine Learning web application built with Gradio and scikit-learn. The model uses Linear Regression to predict a customer's total spend based on demographic and behavioral data.

This project was developed as the advanced optional deployment assignment for the "Python for Data Science, Machine Learning and Artificial Intelligence" course (MSc in BIS-Analytics, University of Athens).

## 📊 Dataset Source & License
* **Source:** The dataset used to train this model is a sample from the marketing_campaign.csv dataset available on Kaggle
